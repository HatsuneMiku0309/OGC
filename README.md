# OGC (Open Generation Chat)

This project use vite (ESModule).

## Environment
* Vue3
* Vite
* tailwindcss 2.x


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### preview dist

```
npm run seve
```


### Lints and fixes files
